# Open Graph Image

Please create an og-image.png file (1200x630px) for social media previews.

## Requirements:

- Size: 1200 x 630 pixels
- Format: PNG or JPG
- Content: Portfolio preview with name, tagline, and visual identity
- File name: og-image.png
- Location: /public/

## Design Tips:

- Include your name and role
- Use brand colors from the theme
- Keep text large and readable
- Avoid too much detail

Once created, the image will automatically be used by social media platforms when sharing your portfolio link.
